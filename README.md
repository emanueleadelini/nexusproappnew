## 🛠️ Guida Rapida Terminale (Per il Push)
Per inviare il codice su GitHub, apri il terminale in basso (**View > Terminal**) e incolla:
```bash
git init
git remote add origin https://github.com/emanueleadelini/Nexuspro.git
git add .
git commit -m "primo push completo"
git branch -M main
git push -u origin main --force
```
